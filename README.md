# 使用方法
1. 打开辅助(给悬浮窗权限)
2. 对准红点，设置起始点和结束点，计算出距离（比如704）
3. 用手机连接USB，cmd执行命令

	`adb shell input swipe 100 100 100 100 704`

（这种方法比较笨，哈哈）
